H.Fugue 調香台 Organ｜App 使用說明

1. 直接開 index.html 可以使用網頁版，但本機 file:// 無法真正安裝成 App。
2. 要「下載成 App」，請把整個資料夾上傳到 GitHub Pages、Netlify、Vercel 或你的網站目錄。
3. 用手機或電腦瀏覽該網址後，Chrome / Edge 可按「下載成 App」或瀏覽器選單「安裝 App」。iPhone / iPad 請用 Safari → 分享 → 加入主畫面。
4. JSON 備份：在配方表按「下載 JSON」。要還原時按「上傳 JSON」，選擇先前下載的檔案。
5. App 圖示目前使用 H.FUGUE ORGAN 文字 logo。若要改成正式品牌圖，替換 icons/logo-192.png 與 icons/logo-512.png 即可。
